# DecodesLabs-Internship
**1. Task Manager System**
A simple Python-based Task Manager application that allows users to add, view, and search tasks efficiently. The project uses JSON file handling for permanent data storage, ensuring persistence even after the program is closed. Users can manage task IDs, titles, and completion status through a menu-driven interface. This project demonstrates core programming concepts such as functions, loops, dictionaries, file handling, and CRUD operations. It is designed for beginners to understand how real-world task management systems work using Python.
**2. Expense Tracker**
The Expense Tracker is a console-based Python application used to record and calculate daily expenses. Users can continuously add expense amounts until they choose to stop, and the program automatically calculates the total spending. The project includes input validation using exception handling to prevent invalid entries. It demonstrates important programming concepts such as loops, accumulators, type conversion, and error handling. This beginner-friendly project helps users understand how financial tracking applications work in real life.
**3. Secure Password Generator**
The Secure Password Generator is a Python-based application that creates strong and random passwords for improved security. Users can enter their desired password length, and the program generates a secure password using a combination of uppercase letters, lowercase letters, numbers, and special symbols. The application ensures password complexity by including at least one character from each category. It also uses randomization and shuffling techniques to enhance security and unpredictability. This project demonstrates the use of Python libraries, loops, conditional statements, lists, and string manipulation for building practical cybersecurity-related applications.
